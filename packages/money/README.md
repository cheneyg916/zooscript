# `money`

> money 操作

## 尝试转数字

```
tryConvertToNumber<number>('123456', 123456)

// 123456
```

## 格式化数字

```
formatNumber(123456, 2)

// 1234.56
```

## 分 转 元

```
fenToYuan('123456', 2)

// 1234.56
```

## 元 转 分

```
yuanToFen('123456')

// 12345600
```

## 千位分隔

```
formatInt('123456')

// 123,456
```
